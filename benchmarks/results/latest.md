> **Root cause of prior misleading results:** The V8 Dockerfile never set `OCJS_OPT` or `OCJS_LTO` env vars, so `compileSources.py` defaulted to `-O0` with no LTO. The YAML `emccFlags` (`-O3 -flto`) only applied at link time, and LTO had no effect since `.o` files lacked LTO bitcode. `wasm-opt -O3` partially compensated, making V8 appear roughly equal to V7 instead of dramatically slower. With proper `-Os + LTO`, V8 is **52-72% faster** and **12% smaller**.
